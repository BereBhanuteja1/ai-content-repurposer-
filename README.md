# 🎬 AttentionX AI - Automated Content Repurposing

## 🚀 Problem
Convert long videos into short viral clips automatically.

---

## 💡 Solution
This project:
- Uploads long videos
- Extracts short clips
- Prepares content for reels/shorts

---

## 🛠 Tech Stack
- FastAPI (Backend)
- Streamlit (Frontend)
- MoviePy (Video Processing)
- Whisper / Gemini (AI - optional)
